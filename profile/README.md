“大部分都是换到其他楼层的客房了……”

    刘昴星放下内线电话之后，有些头疼的说道：“这下有些糟糕了，看来对方是给了12楼原本的客人什么好处，并且帮他们调换了其他房间，恐怕不久之后四十间……阿不，三十九间房都会住满捣乱的家伙了！”

    贵和子的1201号房肯定不会再退，也就是保住了一间……

    “对了，我和其他一些远月酒店的厨师关系不错，能够找到他们帮忙，是不是可以……”刘昴星忽然眼前一亮，用品鉴料理来换人来帮忙的话，对他来说并不难。

    贵和子想了想却说道：“虽然不是不可以……但是你应该也注意到了，每个楼层提供的客房料理服务的菜单是不一样，也就是说在远月酒店，只有122隔壁的厨房与我们是同系统，按照远月度假村的规定，不同系统的料理人不允许私自条换岗位、替班，必须向酒店报备……不到万不得已，你也不希望在实习中留下这样的记录吧？”

    不仅是刘昴星的实习，贵和子与美和子，在年末也有晋升主厨的考核，同样不想给其他人留下一个“需要别人帮忙才能度过困难”的印象。

    “还需要报备吗？”刘昴星眉头一皱，显然也马上将这当做是最后的办法了……

    就在这时，美和子也不请自来，而且这次是直接推门走了进来。

    依旧那身短衣短裤的装扮，刘昴星现在也知道，这身白色的短衣短裤，就是她在厨房时的工作装！

    “爸妈没教过你敲门吗！”贵和子怒道，看来果然和美和子看不过眼。

    “爸妈没教过你，出门在外不要乱得罪人吗！”美和子反唇相讥。
